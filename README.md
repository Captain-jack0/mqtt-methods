# MQTT Methods Collection

- **method1/**: Yöntem 1 açıklaması

Nasıl kullanılır Local de başlatmak Python ile
1. python3 -m venv venv
2. source venv/bin/activate        # macOS/Linux
3. pip install paho-mqtt
4. python ingest.py
